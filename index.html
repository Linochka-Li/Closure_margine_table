Шаг 1 из 4

Таблица маржинальности

Подключите гугл-таблицу напрямую (тогда изменения можно будет записать обратно в неё одной кнопкой) — либо загрузите .xlsx-файл как раньше. Программа найдёт колонки подрядчиков и список проектов.

🔗 Google Таблица
📄 Файл .xlsx

Получить в Google Cloud Console — инструкция в README. Client ID не секретный, его можно вписать в код сайта, чтобы коллеги не искали его сами.

Файл таблицы

Файл → Скачать → Microsoft Excel (.xlsx) из вашей гугл-таблицы, лист текущего месяца.

Шаг 2 из 4

Загрузка сверки

Загружайте файлы по одному — сверку подрядчика (1 или 2 листа), отдельный биллинг DSP, МТС или Директ. Программа сама определит тип файла.

Файл сверки

Поддерживаются: сверка подрядчика (Поток/Кампания/Цена/Показы/Сумма), биллинг DSP (SSP/Impressions/Payout), МТС, Яндекс.Директ.

Автоматически исключать DSP без названия кампании
Строки с реальными показами и суммой, но без привязки к проекту, будут удаляться без вопроса. Если выключить — они попадут в очередь на подтверждение на шаге 3.
